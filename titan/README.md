1. 确保已经安装 docker,根据自己的电脑选择不同的安装方式
2. 执行脚本，注意需要 调整 hash 参数

```shell

curl -sSL "https://raw.githubusercontent.com/ansike/titan/master/titan/create-titan-servers.sh" | bash -s FA7C84E6-4B22-43E3-BC83-40DC09A6BC61
```
